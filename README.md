# dotfiles

## About
Current dotfiles:
* .vimrc
* .tmux.conf
# .shell_aliases

### How to use
* take inspiration and write your own dotfiles
* create symlinks manually from `$HOME` to this repo
* run `./install.sh` (read it first!) to create the symlinks automatically
